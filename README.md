# Dynamic IAM Agent Dashboard

A production-style access management dashboard built with React, Vite, and Tailwind CSS.

## Features

- Login screen with persistent user session
- Sidebar navigation across dashboard modules
- Custom access duration input (minutes, hours, days)
- Live session tracker with countdown and revoke support
- Usage history with search and filters
- Session timeline and admin audit view
- Settings panel (compact mode, animation toggle)
- Local storage persistence for sessions/history/settings

## Tech Stack

- React 18
- Vite 5
- Tailwind CSS 3

## Run Locally

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
